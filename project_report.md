# تقرير شامل عن مشروع "أركان" (Arkan)

منصة **أركان** هي منصة تعليمية برمجية متكاملة ومبتكرة مصممة خصيصاً لتعليم تطوير الويب والبرمجة باللغة العربية بطريقة تفاعلية وممتعة (Gamified Learning).

---

## 1. بنية النظام والتقنيات المستخدمة (Tech Stack)

تم بناء المنصة باستخدام أحدث التقنيات لتقديم أداء سريع وتجربة مستخدم متميزة:

* **الإطار الأساسي (Frontend & Backend):** Next.js 14 (App Router) مع React 18 و TypeScript لضمان كتابة كود منظم وآمن.
* **قاعدة البيانات وإدارة الجداول:** Prisma ORM للتعامل مع قاعدة بيانات PostgreSQL.
* **التصميم والتنسيق:** Tailwind CSS مع Framer Motion للحركات والتأثيرات التفاعلية، و Lucide React للأيقونات.
* **الذكاء الاصطناعي:** دمج نموذج Gemini AI (عبر `@google/generative-ai`) لتوفير مساعد برمجيات ذكي وتفاعلي للطلاب يجيب على أسئلتهم البرمجية بشكل مهيكل واحترافي.
* **الرسومات الإحصائية:** Recharts لعرض تحليلات تقدم الطالب ومستوى نشاطه داخل لوحة التحكم.
* **نظام البريد الإلكتروني:** SMTP عبر Gmail (باستخدام Nodemailer) لإرسال رسائل الترحيب وإعادة تعيين كلمات المرور.
* **التحقق والأمان:** نظام مصادقة مخصص باستخدام JWT (`jose` / `jsonwebtoken`) وتشفير كلمات المرور بـ `bcryptjs` مع نظام لحماية الجلسات وتتبع الأجهزة وسجل تسجيل الدخول.

---

## 2. الميزات الرئيسية للمنصة (Key Features)

### 📚 نظام الدورات والمناهج التعليمية
* تقسيم الدورة إلى وحدات (Modules) ودروس (Lessons).
* دعم تشغيل الفيديوهات البرمجية (عبر Cloudflare Stream أو Vimeo).
* اختبارات تفاعلية قصيرة (Quizzes) بعد كل درس، بالإضافة إلى اختبارات نهائية للدورات.
* نظام تسليم وتقييم الواجبات والمشاريع العملية (Assignments).

### 🎮 الألعاب والتعلم باللعب (Gamification)
تحتوي المنصة على مجموعة من الألعاب البرمجية والمنطقية المبتكرة لتنمية مهارات التفكير البرمجي:
1. **حامي Flexbox (Flexbox Defender):** لعبة لتعلم محاذاة عناصر CSS.
2. **معركة CSS (CSS Battle):** مطابقة تصاميم HTML/CSS بأقل كود ممكن.
3. **ثعبان بايثون (Python Snake):** كتابة أوامر برمجية للتحكم بالثعبان وتفادي العقبات.
4. **مبارزة SQL (SQL Duel):** كتابة استعلامات SQL حقيقية ضد الوقت.
5. **صياد الأنماط Regex (Regex Hunter):** صيد النصوص وتصميم تعبيرات نمطية صحيحة.
6. ألعاب منطقية أخرى مثل: لعبة الذاكرة، مدرب الكتابة السريعة، وألغاز الخوارزميات.
7. **نظام النقاط والمستويات:** كسب نقاط الخبرة (XP) وسلاسل النشاط اليومي (Streak) ومكافآت الإنجازات (Achievements).

### 🤖 مساعد الذكاء الاصطناعي الذكي (Gemini Assistant)
* نظام بحث معزز بالاسترجاع (RAG) يقوم بالبحث في محتوى الدروس وجلب المعلومات الأكثر صلة بسؤال الطالب.
* صياغة الإجابة البرمجية بهيكل تعليمي فائق الترتيب يحتوي على:
  1. تعريف مبسط للمفهوم.
  2. شرح عميق.
  3. أمثلة برمجية متعددة (3-4 أمثلة).
  4. مقارنة مع المفاهيم القريبة لتوضيح الفروق.
  5. الاستخدامات العملية والسيناريوهات الواقعية.
  6. الأخطاء الشائعة التي يقع فيها المبرمجون.
  7. مصادر إضافية وأسئلة تفاعلية لاختبار فهم الطالب.

### 📜 نظام الشهادات التلقائية
* توليد شهادات إتمام الدورات بصيغة PDF وتصميم راقٍ وحفظها تلقائياً وتوليد رمز استجابة سريعة (QR Code) فريد لكل شهادة للتحقق من مصداقيتها وصلاحيتها عبر المنصة.

### 🏛️ لوحة تحكم المعلم (Instructor Portal)
* لوحة تحكم مخصصة لإدارة الطلاب، تتبع حضورهم وتقدمهم، تصحيح الواجبات وتحديد درجات الطلاب، وتوليد كوبونات الخصم والوصول المجاني للدورات (Vouchers).

## 3. بنية قاعدة البيانات ومخطط الجداول (Database Schema)

تحتوي قاعدة البيانات على جداول مصممة بعناية للربط بين الكيانات المختلفة:
* **المستخدمون (User):** لتخزين البيانات الشخصية، الأدوار (طالب، معلم، مدير)، النقاط (XP)، وإنجازات التعلم.
* **المساقات (Course, Module, Lesson):** الهيكل الأساسي للمحتوى التعليمي.
* **التقدم والتقييم (LessonProgress, Enrollment, Quiz, Question, QuizAttempt, Assignment, AssignmentSubmission):** لمراقبة التقدم وحفظ نتائج الاختبارات والواجبات.
* **الأمان والجلسات (DeviceSession, LoginEvent):** لتأمين الحسابات ومتابعة محاولات تسجيل الدخول والأجهزة النشطة.
* **الشهادات (Certificate):** لتخزين بيانات تخرج الطلاب وتوليد الـ QR Codes.
* **الكوبونات والمشتريات (Voucher, BookPurchase, Book):** لإدارة العمليات المالية والوصول للدورات والكتب البرمجية الرقمية المتاحة في مكتبة المنصة.

## 4. متمتطلبات بيئة التشغيل وكيفية البدء

لتشغيل المشروع محلياً، يجب اتباع الخطوات التالية:
1. **تثبيت Node.js:** تأكد من تثبيت نسخة Node.js 18 فما فوق (تم الكشف عن نسخة `v24.11.1` وهي مدعومة بالكامل).
2. **إنشاء ملف البيئة `.env`:** نسخ ملف `.env.example` وتعبئته بروابط الاتصال.
3. **توفير قاعدة بيانات PostgreSQL:** إنشاء قاعدة بيانات محلية أو سحابية ووضع رابطها في ملف `.env`.
4. **تثبيت الحزم وتجهيز بريزما:**
   ```bash
   npm install
   npx prisma generate
   ```
5. **تهيئة قاعدة البيانات:**
   ```bash
   npx prisma db push
   npx prisma db seed
   ```
6. **تشغيل المشروع:**
   ```bash
   npm run dev
   ```
